# Telegram Bot Example

A simple example project demonstrating how to create and run a Telegram bot using Python.  
This repository serves as a base for study and experimentation with Telegram bots.

---

## Getting Started

### 1. Clone the repository

```bash
git clone <repo-url>
cd project-folder
```

### 2. Create a virtual environment

```bash
python -m venv venv
```

### 3. Activate the virtual environment

`Windows`

```ps
.\venv\Scripts\activate
```

`Linux`

```bash
source ./venv/Scripts/activate
```

> To deactivate the virtual environment, simply run `deactivate` in the active terminal.

### 4. Install dependencies

```bash
pip install -r requirements.txt
```

### 5. Run the bot

```bash
python main.py
```
